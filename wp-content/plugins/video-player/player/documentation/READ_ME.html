About product

This is HTML 5 Video Player which allows you to add videos easily to your webpage. You can use player without gallery with only one video, or you can build your own gallery with thumbnails, titles and descriptions. You can add as many videos you want. Player interface is configurable via CSS, so you can easily customize colors and adjust it to your needs. 

Directory and files explanation

Inside the download zip from CodeCanyon you'll find the following structure:



deploy - containes "minified" versions of all scripts (js/css). This means the scripts/css files have been compressed so they load faster.

documentation - containes documentation files

source - containes original versions of all scripts (js/css).Identical to "deploy" folder except nothing is minified. If you wish to change css and javascript files use this folder. After making changes, for faster loading you can minify the CSS/JS files by using the following tools:

JS: http://closure-compiler.appspot.com/home
CSS: http://www.refresh-sf.com/yui/

 

How to use / plugin options

Open index.html file from the "deploy" folder.

Video player only needs few lines of code to get started:

    videoPlayer = $("#video").Video();

Features

  • Gallery ON / OFF (you can use this video player without gallery, or, you can add unlimited number of videos and build your gallery with thumbnails, titles, and descriptions)
  • Click & drag to control gallery (optimized for mobile devices)
  • Mousewheel to control gallery
  • Logo support : you can add your logo as image (.jpg, .png). Logo position can be in bottom-left corner, or bottom-right corner.
  • Preview image for player (if you want your player to be paused when page loads, set "autoplay=off" and preview image will appear)
  • Autohide controls (you can choose after how much seconds player controls will disappear)
  • Hide / show gallery with playlist button (with this functionality you can "stretch" player over gallery to have better view for video, as well as "shrink" player to see gallery menu again to choose video)
  • Autoplay (on/off)
  • Complete volume control
  • Complete share menu control (facebook, twitter, myspace, wordpress, linkedin, flickr, blogger, delicious, mail)
  • Information window with HTML formatted text (it means you can insert links, change fonts, sizes with standard HTML tags)
  • Rewind function (to play your videos from beggining)
  • Tooltip for time
  • Tooltip for volume %
  • Real fullscreen support  (when not supported real fullscreen it switches to full browser support)
  • Play/Pause on click or SPACE key
  • Double click to enter/exit fullscreen 
  • ESCAPE key to exit fullscreen/full browser mode

Video Formats

You will need to provide 3 formats to make player work cross-browsers. For example: if you are browsing with Chrome, chrome can play mp4, ogg/ogv, and webm, so it need just one of the formats, but, if you are browsing with Opera 12.12 (latest version at the moment) you'll need to provide format beside mp4 (.ogv or .webm) to be able to play in opera correctly. That is why all three formats of each video are  necessary.

 

To convert your video to mp4/ogg/webm you can use this online converter.

 

Change theme / customisation

You can use multiple themes, to change theme do the following steps:

 

  • in your HTML code find these two lines where it says "theme1"






<link rel="stylesheet" href="stylesheets/videoPlayer.theme1.css" type="text/css">
<link rel="stylesheet" href="stylesheets/videoPlayer.theme1_Playlist.css" type="text/css" media="screen"/>

 

  • make changes to "theme2"






<link rel="stylesheet" href="stylesheets/videoPlayer.theme2.css" type="text/css">
<link rel="stylesheet" href="stylesheets/videoPlayer.theme2_Playlist.css" type="text/css" media="screen"/>

 

  • all done, you have applied theme 2 to your player!

 

 

Beside that you can also combine separately theme for player and theme for playlist like shown in the following example:







<link rel="stylesheet" href="stylesheets/videoPlayer.theme2.css" type="text/css">
<link rel="stylesheet" href="stylesheets/videoPlayer.theme3_Playlist.css" type="text/css" media="screen"/>

 

 

 

If you want to change some color/icon/font by your own, you can do that in for example "videoPlayer.theme1.css"  file for skin1.

 

For example change color of progress bar and turn off radius corners:

.videoPlayer .controls .videoTrack .videoTrackProgress{
position: absolute;
height:5px;
background: #46e5eb;                 /*change color to white: #ffffff */
-webkit-border-radius: 8px;        /*delete this line or enter value 0px to*/
border-radius: 8px;                       /*delete this line or enter value 0px to*/
}

 

 

Multiple instances of player

You can have multiple players per page. Every player can have it's own settings. Add the following in your HTML document:

Example - 2 player in one page:











<script type="text/javascript" charset="utf-8">
 $(document).ready(function($)
 {
    videoPlayer = $("#video").Video({
    });
   //add this for player 2
    videoPlayer = $("#video2").Video({
    });
 });

 </script>

<body>
  <div id="video"></div>
  //add this for player 2:
  <div id="video2"></div>
</body>

Done! Now you should be able to see 2 player in your HTML page.
