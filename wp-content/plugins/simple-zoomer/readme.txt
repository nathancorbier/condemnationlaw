== Description ==
This plugin will enable increase the font size of the page for better reading experience
Once we activate this plugin, it will automatically display a widget in "Available Widgets" section.



== Installation ==
1. Download the file
2. Upload and extract the file the copy the folder into our plugin location like (wp-content/plugins/simple-zommer)
3. Activate the plugin from admin side.
4. Then drag the widget 'Font Zoomer' on sidebar after adding the title save the widget. 
5. Once we add the widget we will get the out put on side bar which ever you place the widget.
